Importing

Workspace provides a library for use with the Swift Package Manager.

Simply add Workspace as a dependency in Package.swift:

let package = Package( name: "MyPackage", dependencies: [ .package( name: "Workspace", url: "https://github.com/SDGGiesbrecht/Workspace", .upToNextMinor(from: Version(0, 33, 3)) ), ], targets: [ .target( name: "MyTarget", dependencies: [ .product(name: "WorkspaceConfiguration", package: "Workspace"), ] ) ] )

The module can then be imported in source files:

import WorkspaceConfiguration
