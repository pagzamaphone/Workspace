Instance Property

serveFromGitHubPagesBranch

Whether or not to automatically serve the documentation using GitHub Pages.

Declaration

var serveFromGitHubPagesBranch: Bool { get set }

Discussion

By setting this to true, projects with continuous integration management active can avoid checking generated files into the main branch.

Workspace will only generate documentation in continuous integration and stop generating it locally. (If needed for coverage checks, Workspace may still do so in a temporary directory.) The generated documentation will be automatically published to GitHub Pages via the gh-pages branch, making the docs directory unnecessary.
