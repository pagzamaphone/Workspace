Einführung

Workspace stellt eine Bibliotek für zur Verwendung mit dem Swift‐Paketenverwalter (Swift Package Manager) bereit.

Workspace einfach als Abhängigkeit (dependency) in Package.swift hinzufügen:

let paket = Package( name: "MeinePaket", dependencies: [ .package( name: "Workspace", url: "https://github.com/SDGGiesbrecht/Workspace", .upToNextMinor(from: Version(0, 33, 3)) ), ], targets: [ .target( name: "MeinZiel", dependencies: [ .product(name: "WorkspaceConfiguration", package: "Workspace"), ] ) ] )

Das Modul können dann in Quelldateien eingeführt (imported) werden:

import WorkspaceConfiguration
