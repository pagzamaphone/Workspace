Typ‐Eigenschaft

selbeZeile

Dieser Geltungsbereich gilt für Abdeckungsbereiche, die auf der selbe Zeile beginnen als das Zeichen.

Festlegung

static var selbeZeile: Geltungsbereich { get }

Einzelheiten

Dieser Geltungsbereich ist nützlich für Funktionen wie behaupten, die untestbare Diagnosemeldungen haben:

behaupten(x == y, "Es gibt ein Problem: \(problem)") // ↑↑↑↑↑↑ // Die Zeichenketteninterpolation kann nicht mit Testen abgedeckt werden... // ...aber das „behaupten“‐Zeichen erlaubt es als Ausnahme.

Dieser Geltungbereich ist auch von dem allgemeinen Ausnahme verwendet, @ausnahme(zu: tests).

func untestbareFunktion() { // @ausnahme(zu: teste) // Die Ausnahme gilt hier. }
