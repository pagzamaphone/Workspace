Typ‐Eigenschaft

vorstehendeZeile

Dieser Geltungsbereich gilt für Abdeckungsbereiche, die auf der Zeile vor dem Zeichen beginnen.

Festlegung

static var vorstehendeZeile: Geltungsbereich { get }

Einzelheiten

Dieser Geltungsbereich ist nützlich für Funktionen wie preconditionFailure, die sich in untestbare Quellzweigen befinden:

guard let x = y else { // ← Der ungetesteter Bereich fängt beim Klammer an... voraussetzungsfehlschlag("Das soll nie geschehen.") // ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑ // ...aber das „voraussetzungsfehlschlag“‐Zeichen erlaubt es als Ausnahme. } // Solche Zeichen gelten auch am der selben Zeile, damit folgende Abschlüsse erlaubt werden: guard let x = y else { voraussetzungsfehlschlag("Das soll nie geschehen.") }
