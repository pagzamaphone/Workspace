Merkmale

  • Stellt gründliche Prüfungen bereit:

    • Testabdeckung

    • Übersetzerwarnungen

    • Dokumentationsabdeckung

    • Beispielprüfung

    • Stilkorrekturlesen

    • Erinnerungen

    • Einrichtung von fortlaufeden Einbindung (GitHub Vorgänge)

  • ErstelltDokumentation von Programierschnittstellen.

  • Automatisiert Quellinstandhaltung:

    • Einbau von Ressourcen

    • Geerbte Dokumentation

    • Erstellung von Xcode‐Projekte

  • Automatisiert quelloffene Nebensachen:

    • Dateivorspänne

    • Lies‐mich‐Dateien

    • Lizenzhinweise

    • Mitwirkungsanweisungen

  • Für Verwendung neben dem Swift Package Manager vorgesehen.

  • Verwaltet Projekte für macOS, Windows, Netz, Linux, tvOS, iOS, Android und watchOS.

  • Konfigurierbar
